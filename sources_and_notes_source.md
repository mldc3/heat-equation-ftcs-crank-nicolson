# Sources and Notes

This repository is based on computational physics coursework and personal modelling notes for the subject *Introducción a la Modelización* at the University of Alicante.

The project corresponds to the study of parabolic partial differential equations, with particular emphasis on the one-dimensional heat equation.

The theoretical material used in the documentation includes:

- classification of partial differential equations,
- finite-difference discretization,
- the one-dimensional heat equation,
- physical derivation from a conservation law and Fourier flux,
- FTCS explicit scheme,
- Crank-Nicolson implicit/semi-implicit scheme,
- matrix formulation of finite-difference operators,
- Dirichlet boundary conditions,
- Von Neumann stability analysis,
- variable diffusion coefficient $D(x)$,
- conservative flux discretization,
- numerical error analysis,
- runtime comparison between explicit and implicit methods.

The repository is a cleaned computational-physics version of the original practice code. It is intended as a professional portfolio project showing numerical modelling, scientific Python programming and interpretation of PDE simulations.

No claim is made that this repository is a general-purpose production PDE solver. Its purpose is educational and demonstrative: to connect theory, implementation and numerical diagnostics in a clear and reproducible way.
